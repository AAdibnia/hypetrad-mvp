HypeTrade: Phase 1 (MVP) Backlog

A sprint-ready backlog for building the core product in the first month.

Backlog (Prioritized)

EPIC: Core Trading

HT-9

As a user, I want my portfolio dashboard to show my total portfolio value and my overall unrealized profit or loss so I can track my performance.

Acceptance Criteria

  • Total value is sum of cash and all holdings' current value.
  • P/L is calculated correctly and color-coded (green/red).
EPIC: Core Trading

HT-15

As a user, I want to see the current price and profit/loss for each individual purchase.

Acceptance Criteria

  • Holdings table has columns for Purchase Price, Current Price, and P/L.
  • P/L is calculated as (Current Price - Purchase Price) * Quantity.
  • P/L is color-coded (green for profit, red for loss).

Acceptance Criteria
